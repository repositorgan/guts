# Document Summarizer

**Lightweight document summarization service designed for:**

> Legal teams reviewing contracts and evidence
> Educators improving reading comprehension
> Analysts needing fast document overviews
> Media outlets seeking consolidation of topics

## Features
> Sentence-scoring summarization
> Clean HTTP API
> Auto-generated documentation
> Easy to deploy and extend

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
